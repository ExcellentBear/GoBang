# GoBang
gobang with a naive tranditional ai

# 简单的五子棋
* 五子棋程序随处可见，但都晦涩难懂。 
* 查看了多种五子棋AI代码，js,c++, java, golang什么的都有, 最后捋出了个简单的AI版本

# AI部分代码主要为4个函数，总共不到200行
* 界面与AI算法分离，可以忽略界面文件
* 采用了c++的stl, 不涉及复杂自定义结构可读性更强.
* 代码量少，快速看懂

# 附带界面, 以及编译的二进制程序
* 界面简单，棋盘依托qwidgettable构建，暴力快捷
* 可以体验弱智的AI是否达到预期

# 作业、学习 均可作为参考
* 总共才几个文件，删删增增又是个程序
* AI部分,重要注释均已添加
